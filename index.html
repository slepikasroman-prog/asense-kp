Инструкция по работе
1. Клиент
Выберите сегмент объекта (ресторан / отель / СПА / глэмпинг) — от этого зависят нормы расхода продуктов.
Заполните карточку клиента: название, контакт, город, адрес. Незаполненные поля просто не отображаются в КП.
Укажите параметры объекта: гостей в день и рабочих дней в месяц — приложение автоматически рассчитает нужное количество канистр.
Поле «Скидка / наценка %» пересчитывает все цены в КП пропорционально.
2. Продукты
Поставьте галочки напротив нужных позиций.
У каждого продукта есть выпадающий список ароматов — выбранный аромат попадает в КП рядом с названием продукта.
Мыло-скраб — аромат фиксированный (Vetiver & Rosemary), выбор не нужен.
Для канистр 5л количество рассчитывается автоматически по норме расхода. Для ароматики, миниатюр и оборудования — вводите вручную.
Раздел «Дополнительные позиции» — добавьте любую позицию вручную. Цена 0 или 1 ₽ отобразится в КП как «Бесплатно».
3. Ароматы
Выберите ароматы для итогового примечания в КП.
Косметика — основная линейка (Pure Cannabis, Virgin Lemongrass и др.).
Ароматизация помещений — отдельная линейка диффузоров.
SPA-ароматы — специальная SPA edition (White Lotus, Hamam Lover, Meditation Session).
4. Предпросмотр и выгрузка
Предпросмотр — показывает готовое КП как оно будет выглядеть для клиента.
PDF / Печать — открывает окно печати браузера. Выберите «Сохранить как PDF» чтобы получить файл.
Excel — выгружает два листа: КП с фирменным оформлением + расчёт расхода по трём сценариям загрузки (50 / 70 / 100%).
5. История КП
Нажмите «Сохранить» в верхней панели — КП сохранится в браузере с датой, суммой и именем клиента.
В истории можно: открыть КП в одно нажатие, посмотреть предпросмотр в новой вкладке, удалить.
Бэкап JSON — скачайте полный бэкап всех КП и изменённых цен на компьютер.
Загрузить бэкап — восстановите данные из JSON файла на любом устройстве. Дубли не создаются.
⚠ Данные хранятся только в браузере на этом устройстве. Регулярно делайте бэкап!
6. Прайс
Здесь можно изменить цены для конкретного клиента или установить персональные цены глобально.
Цена меняется сразу при вводе — сохранять отдельно не нужно.
Изменённые цены подсвечиваются золотым, рядом показывается базовая цена из прайса.
Сбросить — возвращает все цены к базовым из прайса HoReCa.
Цены сохраняются в браузере и применяются ко всем новым КП.
Полезные советы
Сегмент влияет на автозаполнение «Гостей в день» — при переключении значение обновится.
Для разных клиентов с разными ценами используйте поле «Скидка / наценка %» — быстрее чем менять прайс.
КП автоматически обновляется при изменении любого параметра — предпросмотр всегда актуален.
На GitHub Pages приложение работает полностью в браузере — интернет нужен только для открытия страницы.
